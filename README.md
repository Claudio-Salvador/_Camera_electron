# Backend-NodeJS

## Stacks
* Nodejs
* Express
* JS
* MySQL
* SOLID Principles

## Clone & Open
```
    git clone https://github.com/Claudio-Salvador/_Camera_electron.git
    cd _camera_electron
```
## Install Node 
```
  Download in official web site: https://nodejs.org/en
```

## Install & Run
```
  yarn install or npm install or pnpm install
  yarn dev or npm run dev or pnpm run dev
```
